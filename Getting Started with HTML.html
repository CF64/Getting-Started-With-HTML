Important Concepts

About The Web

Components of The Web include:
HTTP- The main protocol of the web
Servers- Computers that host the files that make up the web
Internet- The world's largest computer network
Browser- A program that runs on your computer to display files found on the web e.g. Chrome, Firefox,and Edge.
Everything comes together when your computer conected to the internet, and you use a browser like Chrome or Safari to make a request using the HTTP protocol from YouTubeservers to display a cat video to your screen.

HTML

HTML stands for Hypertext Markup Language. HTML documents are the are the heart of the web. HTML is made up of Text Content which is "what you you see", Markup, which is wht the text looks like. It is also made up of references to other documents such as images, videos and links to other pages.

Tags and Elements

HTML Markup is made made using Tags.Elements usually contain an openinng Tag, content, and an end tag. This video explains more about Tags and Elements

HTML Attributes

Atributes are elements that add additional information to the browser. For example, the anchor tag takes an attribute called href that defines a URL to link to.

Why Computers are Stupid

Computers interpret Language literally which is why they are stupid. It is up to the programmer to make sure there are no mistakes.

Inline vs Block Elements

Inline elements affect only contents in a line, while Block elements create a box around the content. This video, further explains the distictions between inline and block elements.

Document Structure

The Document Structure of an HTML consist of a DOCTYPE which defines the type of document, then an openning HTML tag, followed by a head tag that defines the metadata for the web page, and a title, then a body tag which would contain all the contents of the web page.
